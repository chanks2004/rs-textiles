# Cotiza Playeras

Sitio web profesional para un intermediario textil que conecta marcas con fabricantes de playeras, sudaderas y gorras personalizadas.

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Estructura

- `src/app/` — Páginas y layouts (App Router)
- `src/components/` — Header, Footer y componentes UI reutilizables
- Diseño: fondo blanco, tipografía negra/gris oscuro, color de acento verde (#0f766e)

## Páginas

- **Inicio** — Hero, cómo funciona, productos, estadísticas, testimonios, CTA
- **Playeras** — Tipos de tela, métodos de impresión, tiempos, CTA
- **Sudaderas** — Tipos, materiales, producción, CTA
- **Gorras** — Tipos de gorras, opciones de bordado, CTA
- **Cómo funciona** — Cotización, muestras, producción, entrega
- **Cotizar** — Formulario de solicitud de cotización
- **Nosotros** — Descripción del negocio
- **Contacto** — WhatsApp, email, teléfono
